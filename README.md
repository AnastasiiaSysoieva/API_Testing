# API_Testing

This repository contains examples of API testing performed using Postman and publicly available APIs from Swagger documentation.

The goal of this project is to demonstrate practical skills in API testing, request validation, and response verification.

---

# Tools Used

- Postman
- Swagger
- REST API
- JSON
- GitHub

---

# Testing Scope

The following API testing activities were performed:

- Sending HTTP requests
- Verifying response status codes
- Validating response body
- Testing request parameters
- Testing headers
- Basic negative testing

---

# HTTP Methods Tested

- GET – retrieve data
- POST – create new resources
- PUT – update existing resources
- DELETE – remove resources


---

# Example Test Checks

Examples of validations performed during testing:

- Status code verification (200, 201, 400, 404)
- Response time validation
- JSON structure validation
- Field value verification

---

# Purpose

This project demonstrates manual API testing skills and experience working with REST APIs using Postman.
